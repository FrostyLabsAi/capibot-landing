# 대시보드 개요

Mission Control은 CapiBot을 관리하기 위한 명령 센터입니다. 이 가이드는 인터페이스와 주요 탐색을 안내합니다.

## Mission Control 접근

**접근:** 브라우저를 열고 `http://localhost:3002`로 이동 (또는 구성된 URL)

**로그인:** 자격 증명을 사용하세요 (테스트용 기본값: admin / admin)

## 주요 레이아웃

Mission Control은 세 가지 주요 영역으로 구성됩니다:

```
┌─────────────────────────────────────────────────────────────┐
│  헤더 바 (검색, 알림, 설정, 시계)                            │
├──────────────┬──────────────────────────────┬───────────────┤
│              │                              │               │
│   Nav Rail   │       메인 콘텐츠            │  라이브 피드   │
│   (아이콘)   │      (선택에 따라 변경)      │  (활동        │
│              │                              │   스트림)     │
│              │                              │               │
└──────────────┴──────────────────────────────┴───────────────┘
```

## 탐색 레일

왼쪽 사이드바에는 패널 간 전환을 위한 아이콘이 있습니다:

### 핵심 그룹

- **개요** — 통계와 상태가 있는 대시보드
- **에이전트** — AI 분대 관리
- **스킬** — 에이전트 스킬 탐색 및 설치
- **작업** — 작업 관리를 위한 칸반 보드

### 기업

- **기업** — AI 기업 관리

### 관찰

- **활동** — 실시간 이벤트 스트림
- **로그** — 시스템 로그 및 디버깅
- **메모리** — 에이전트 메모리와 지식 탐색
- **분석** — 성과 지표
- **파일** — 파일 시스템 브라우저

### 자동화

- **Cron** — 예약된 작업
- **웹훅** — 수신 및 발신 웹훅
- **알림** — 알림 규칙과 알림

### 통합

- **통합** — GitHub, Composio 연결
- **GitHub 동기화** — GitHub 이슈 가져오기

### 관리자

- **설정** — 시스템 구성
- **감사 추적** — 보안 및 접근 로그
- **사용자** — 사용자 관리 (다중 사용자인 경우)

**모바일:** 작은 화면에서 탐색 레일은 우선 항목이 있는 하단 바가 됩니다.

## 헤더 바

화면 상단에 표시되는 것:

### 왼쪽

- **현재 패널 이름** — 현재 위치
- **빵조각** — 탐색 경로 (하위 페이지에서)

### 중앙

- **검색** (Cmd/Ctrl + K) — 작업, 에이전트 등에서 빠르게 찾기

### 오른쪽

- **알림** — 안 읽은 개수가 있는 벨 아이콘
- **라이브 피드 토글** — 오른쪽 사이드바 표시/숨기기
- **설정** — 설정에 빠르게 접근
- **테마 토글** — 라이트/다크 모드
- **디지털 시계** — 현재 시간
- **게이트웨이 상태** — 연결 표시기 (녹색 = 연결됨)

## 개요 패널

통계와 상태가 있는 랜딩 대시보드:

### 메트릭 카드

한 눈에 주요 숫자:

- **활성 세션** — 현재 진행 중인 대화
- **온라인 에이전트** — 몇 명의 에이전트가 실행 중인지
- **실행 중인 작업** — 진행 중인 작업
- **오류 (24시간)** — 지난 24시간 동안의 문제

### 시스템 상태

- **게이트웨이 상태** — WebSocket 연결
- **메모리 사용량** — 시스템 리소스
- **데이터베이스** — 연결 및 크기
- **가동 시간** — 얼마나 오래 실행 중인지

### 최근 활동

타임스탬프가 있는 마지막 몇 가지 이벤트

### 빠른 작업

- 새 기업 생성
- 에이전트 생성
- 작업 추가
- 알림 보기

## 라이브 피드 (오른쪽 사이드바)

모든 시스템 활동의 실시간 스트림:

### 보이는 것

- 작업 생성/업데이트/완료
- 에이전트 생성/활성화/유휴
- 메시지 전송/수신
- 파일 생성/수정
- 알림 트리거

### 기능

- **자동 스크롤** — 상단에 최신 이벤트
- **필터** — 특정 이벤트 유형만 표시
- **검색** — 특정 이벤트 찾기
- **일시정지** — 읽기 위해 자동 스크롤 중지

### 이벤트 유형

- 📝 작업 이벤트 (생성, 업데이트, 완료)
- 🤖 에이전트 이벤트 (생성, 상태 변경)
- 💬 메시지 이벤트 (전송, 수신)
- 📁 파일 이벤트 (생성, 편집)
- 🔔 알림 이벤트 (트리거, 해결)
- 🏢 기업 이벤트 (생성, 단계 변경)

**토글:** « 버튼을 클릭하여 라이브 피드를 숨기거나 표시하세요.

## 일반적인 작업

### 키보드 단축키

| 단축키        | 작업              |
| ------------ | ----------------- |
| Cmd/Ctrl + K | 전역 검색         |
| Cmd/Ctrl + / | 단축키 도움말 표시 |
| Escape       | 모달/패널 닫기    |

### 전역 검색

Cmd/Ctrl + K를 눌러 검색:

- 작업 (제목, ID, 담당자별)
- 에이전트 (이름, 역할별)
- 기업 (이름별)
- 파일 (경로별)
- 메모리 항목

### 생성하기

대부분의 패널에는 오른쪽 상단에 "+" 버튼 또는 "새" 버튼이 있습니다.

### 탐색 패턴

**드릴 다운:**

```
개요 → 메트릭 클릭 → 세부 정보 보기
에이전트 → 에이전트 클릭 → 에이전트 상세 페이지
기업 → 기업 클릭 → 기업 상세
```

**컨텍스트 작업:**

- 항목을 마우스 오른쪽 클릭하여 컨텍스트 메뉴
- 빠른 작업을 위해 마우스 오버
- 다른 측면을 위한 탭이 있는 상세 보기

## 패널별 레이아웃

### 목록 보기 (에이전트, 기업, 작업)

- **필터 바** — 상단에 검색 및 필터
- **목록** — 스크롤 가능한 항목
- **일괄 작업** — 여러 항목 선택

### 상세 보기 (에이전트, 기업, 작업 상세)

- **헤더** — 이름, 상태, 주요 정보
- **탭** — 다른 측면 (개요, 활동, 설정)
- **작업** — 편집, 삭제 등

### 보드 보기 (작업 칸반)

- **열** — 상태 열
- **카드** — 끌어서 놓을 수 있는 작업 카드
- **도구 모음** — 필터 및 보기 옵션

### 양식 보기 (생성/편집)

- **섹션** — 그룹화된 필드
- **유효성 검사** — 실시간 오류 확인
- **작업** — 저장, 취소, 삭제

## 모바일 경험

모바일 기기에서:

- **탐색 레일**이 하단 탭 바가 됨
- **라이브 피드**가 슬라이드 아웃 패널로 이동
- **테이블**이 카드가 됨
- **양식**이 수직으로 쌓임
- **모달**이 전체 화면이 됨

## 사용자 정의

### 테마

- **라이트 모드** — 깔끔하고 전문적인
- **다크 모드** — 눈에 편안한
- **자동** — 시스템 설정 따름

헤더 또는 설정 패널에서 토글하세요.

### 밀도

- **편안함** — 더 많은 간격 (기본값)
- **컴팩트** — 화면에 더 많은 정보

### 알림

설정에서 구성:

- 어떤 이벤트가 알림을 주는지
- 알림 채널 (인앱, Telegram 등)
- 방해 금지 시간

## 도움 받기

### 인앱 도움말

- **툴팁** — 아이콘과 버튼을 마우스 오버
- **빈 상태** — 섹션이 비어 있을 때 도움이 되는 프롬프트
- **컨텍스트 힌트** — 사용자가 하는 일에 기반한 팁

### 이 문서

- 탐색 레일에서 연결된 전체 가이드 (활성화된 경우)
- 이 문서 검색
- 특정 기능 가이드 확인

## 일반적인 워크플로우

### 하루 시작

1. 시스템 상태를 위해 개요 확인
2. 알림 검토
3. 밤사이 활동을 위해 라이브 피드 검색
4. 긴급 항목을 위해 작업 확인
5. Human Review 항목에 응답

### 새 기업 생성

1. 탐색 레일에서 기업 클릭
2. "새 기업" 버튼 클릭
3. 브리핑 양식 작성
4. 제출하고 CEO 계획 대기
5. 계획 검토 및 승인

### 에이전트 작업 확인

1. 탐색 레일에서 작업 클릭
2. 담당자 또는 상태별 필터
3. Review 열의 작업 검토
4. 승인하거나 변경 요청
5. 활동 탭에서 세부 정보 확인

### 시스템 상태 모니터링

1. 개요 패널 메트릭 확인
2. 게이트웨이 상태가 녹색인지 확인
3. 오류 수 검토
4. 문제가 있으면 로그 확인
5. 필요하면 설정 조정

## 효율을 위한 팁

1. **검색 사용** — Cmd/Ctrl + K가 탐색보다 빠름
2. **라이브 피드 보기** — 발생하는 대로 활동 보기
3. **키보드 단축키** — 일반적인 것들을 배우세요
4. **중요한 것 고정** — 주요 에이전트/기업을 북마크
5. **알림** — 중요한 것을 알도록 구성
6. **정기 체크인** — 매일 대시보드 검토

## 문제 해결

**페이지가 로드되지 않음:**

- CapiBot 코어가 실행 중인지 확인 (포트 3000)
- Mission Control 포트 확인 (3002)
- 브라우저 캐시 지우기

**게이트웨이 연결 끊김:**

- WebSocket 포트 3001 확인
- 방화벽 설정 확인
- CORS 구성 확인

**성능 저하:**

- 데이터베이스 연결 확인
- 활성 세션 수 검토
- 오래된 로그와 활동 지우기
