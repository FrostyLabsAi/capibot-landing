### Phase Badges

Visual indicators of company state:

- 🟡 **Briefing** — Setting up, waiting for plan
- 🔵 **Planning** — CEO creating business plan
- 🟢 **Approved** — Plan approved, ready to operate
- 🟣 **Operating** — Actively running
- ⚪ **Archived** — Completed or paused

## Creating a Company

### Step-by-Step

1. **Click "+ New Company"**

2. **Fill Company Details:**

   ```
   Name: Q1 Marketing Campaign
   Mission: Launch product awareness campaign
            targeting enterprise customers

   CEO: [Atlas ▼]
   Mode: [Managed ▼]
   Budget: $100/month
   ```

3. **Set Goals:**

   ```
   Goal 1: Publish 4 blog posts
   Goal 2: Generate 50 qualified leads
   Goal 3: Create product demo video
   ```

4. **Submit Briefing**

5. **CEO Creates Plan** (usually within minutes)

6. **Review Plan** — See next section

## Switching Management Modes

### Managed → Autonomous

**When ready:**

1. Go to company Overview tab
2. Click "Mode: Managed" dropdown
3. Select "Autonomous"
4. Confirm: "Are you sure? CEO will make decisions without approval."

**The CEO can then:**

- Hire agents without asking
- Create tasks automatically
- Execute plans independently
- Report progress periodically

### Autonomous → Managed

**When you want oversight:**

1. Go to company Overview tab
2. Click "Mode: Autonomous" dropdown
3. Select "Managed"
4. All future decisions require approval

## Archiving Companies

When a company completes its mission:

1. **Click "Archive" button**
2. **Choose option:**
   - **Archive Only** — Preserve everything, stop operations
   - **Archive & Transfer** — Move unfinished tasks elsewhere
   - **Delete** — Remove completely (use carefully!)

3. **Company moves to Archived state**
   - Agents released (can be reused)
   - Tasks frozen
   - Knowledge base preserved
   - Can be reactivated later

## Best Practices

### Setting Up Companies

1. **Clear Mission** — Specific, achievable goals
2. **Right CEO** — Choose based on domain expertise
3. **Realistic Budget** — Start small, scale up
4. **Defined Timeline** — 2-8 weeks typical
5. **Measurable Goals** — Specific targets

### Operating Companies

1. **Review Plans Carefully** — Sets foundation for everything
2. **Monitor Early** — Watch first few tasks closely
3. **Build Knowledge Base** — Save important documents
4. **Check Costs Weekly** — Avoid surprises
5. **Communicate Regularly** — Use Chat tab for updates

### Approving Work

1. **Timely Reviews** — Don't let tasks sit in Human Review
2. **Clear Feedback** — Specific requests for changes
3. **Document Decisions** — Add notes to approvals
4. **Trust but Verify** — Spot-check autonomous work

## Common Workflows

### Launching a Campaign

1. Create "March Campaign" company
2. Assign Atlas as CEO
3. Set goals: "Generate 100 leads"
4. Atlas submits plan with team structure
5. Approve plan
6. Atlas hires Echo (Strategy), Nova (Content), Pixel (Design)
7. Tasks appear on board
8. Review deliverables in Knowledge tab
9. Approve final content
10. Archive company when done

### Running Ongoing Support

1. Create "Customer Support" company
2. Set permanent status (no end date)
3. Atlas hires Support agents
4. Connect to support channels
5. Agents handle tickets as tasks
6. Build knowledge base from solutions
7. Review monthly analytics
8. Keep running indefinitely

## Troubleshooting

**Company Stuck in Planning:**

- CEO may need more information
- Check Chat tab for questions
- Provide clarification
- Or edit briefing and resubmit

**Tasks Not Moving:**

- Check if agents are Online
- Verify tasks aren't Blocked
- Look at Activity tab for errors
- May need to spawn more agents

**Costs Too High:**

- Review Costs tab for breakdown
- Consider cheaper LLM models
- Reduce agent count
- Set stricter budget limits

**Approvals Backing Up:**

- Review Approvals tab regularly
- Set aside time for reviews
- Consider switching to Autonomous
- Or hire fewer agents
