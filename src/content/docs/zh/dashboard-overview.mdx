# 仪表板概览

Mission Control 是管理 CapiBot 的指挥中心。本指南将带您了解界面和主要导航。

## 进入 Mission Control

**访问：** 打开浏览器并访问 `http://localhost:3002`（或您配置的 URL）

**登录：** 使用您的凭据（测试默认：admin / admin）

## 主布局

Mission Control 有三个主要区域：

```
┌─────────────────────────────────────────────────────────────┐
│  标题栏（搜索、通知、设置、时钟）                              │
├──────────────┬──────────────────────────────┬───────────────┤
│              │                              │               │
│   导航栏     │       主内容区                │  实时动态     │
│   （图标）   │      （根据选择变化）          │  （活动流）    │
│              │                              │               │
└──────────────┴──────────────────────────────┴───────────────┘
```

## 导航栏

左侧边栏包含用于切换面板的图标：

### 核心组

- **Overview** — 带有统计和状态的仪表板
- **Agents** — 管理您的 AI 团队
- **Skills** — 浏览和安装 Agent 技能
- **Tasks** — 工作管理的看板

### 公司

- **Companies** — AI 公司管理

### 观察

- **Activity** — 实时事件流
- **Logs** — 系统日志和调试
- **Memory** — 浏览 Agent 记忆和知识
- **Analytics** — 性能指标
- **Files** — 文件系统浏览器

### 自动化

- **Cron** — 定时任务
- **Webhooks** — 入站和出站 Webhook
- **Alerts** — 告警规则和通知

### 集成

- **Integrations** — GitHub、Composio 连接
- **GitHub Sync** — 导入 GitHub 问题

### 管理

- **Settings** — 系统配置
- **Audit Trail** — 安全和访问日志
- **Users** — 用户管理（如果多用户）

**移动端：** 在小屏幕上，导航栏变为底部栏，显示优先级项目。

## 标题栏

屏幕顶部显示：

### 左侧

- **当前面板名称** — 您在何处
- **面包屑** — 导航路径（在子页面中）

### 中间

- **搜索**（Cmd/Ctrl + K）— 在任务、Agent 等中快速查找

### 右侧

- **通知** — 铃铛图标，显示未读数量
- **实时动态切换** — 显示/隐藏右侧边栏
- **设置** — 快速访问设置
- **主题切换** — 浅色/深色模式
- **数字时钟** — 当前时间
- **网关状态** — 连接指示器（绿色 = 已连接）

## Overview 面板

显示关键数字的着陆仪表板：

### 指标卡片

一目了然的关键数字：

- **Active Sessions** — 当前正在进行的对话
- **Agents Online** — 多少 Agent 正在运行
- **Tasks Running** — 正在进行的工作
- **Errors (24h)** — 最近 24 小时的问题

### 系统健康

- **Gateway Status** — WebSocket 连接
- **Memory Usage** — 系统资源
- **Database** — 连接和大小
- **Uptime** — 运行时间

### 最近活动

最近的几条事件，带时间戳

### 快速操作

- 创建新公司
- 生成 Agent
- 添加任务
- 查看通知

## 实时动态（右侧边栏）

所有系统活动的实时流：

### 您看到的内容

- 任务创建/更新/完成
- Agent 生成/激活/空闲
- 消息发送/接收
- 文件创建/修改
- 告警触发

### 功能

- **自动滚动** — 最新事件在顶部
- **筛选** — 仅显示某些事件类型
- **搜索** — 查找特定事件
- **暂停** — 停止自动滚动以便阅读

### 事件类型

- 📝 任务事件（创建、更新、完成）
- 🤖 Agent 事件（生成、状态更改）
- 💬 消息事件（发送、接收）
- 📁 文件事件（创建、编辑）
- 🔔 告警事件（触发、解决）
- 🏢 公司事件（创建、阶段更改）

**切换：** 点击 « 按钮隐藏/显示实时动态。

## 常见操作

### 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| Cmd/Ctrl + K | 全局搜索 |
| Cmd/Ctrl + / | 显示快捷键帮助 |
| Escape | 关闭模态框/面板 |

### 全局搜索

按 Cmd/Ctrl + K 搜索：

- 任务（按标题、ID、负责人）
- Agent（按名称、角色）
- 公司（按名称）
- 文件（按路径）
- 记忆条目

### 创建内容

大多数面板在右上角有一个 "+" 按钮或 "New" 按钮来创建新项目。

### 导航模式

**深入：**

```
Overview → 点击指标 → 查看详情
Agents → 点击 Agent → Agent 详情页
Companies → 点击公司 → 公司详情
```

**上下文操作：**

- 右键点击项目显示上下文菜单
- 悬停显示快速操作
- 详情视图有不同方面的标签

## 面板特定布局

### 列表视图（Agents、Companies、Tasks）

- **筛选栏** — 顶部有搜索和筛选
- **列表** — 可滚动的项目
- **批量操作** — 选择多个项目

### 详情视图（Agent、Company、Task 详情）

- **标题** — 名称、状态、关键信息
- **标签** — 不同方面（Overview、Activity、Settings）
- **操作** — 编辑、删除等

### 面板视图（Tasks 看板）

- **列** — 状态列
- **卡片** — 可拖动的任务卡片
- **工具栏** — 筛选和视图选项

### 表单视图（创建/编辑）

- **部分** — 分组的字段
- **验证** — 实时错误检查
- **操作** — 保存、取消、删除

## 移动端体验

在移动设备上：

- **导航栏** 变为底部标签栏
- **实时动态** 移动到滑出面板
- **表格** 变为卡片
- **表单** 垂直堆叠
- **模态框** 变为全屏

## 自定义

### 主题

- **浅色模式** — 干净、专业
- **深色模式** — 护眼
- **自动** — 跟随系统偏好

在标题栏或 Settings 面板中切换。

### 密度

- **舒适（Comfortable）** — 更多间距（默认）
- **紧凑（Compact）** — 屏幕上显示更多信息

### 通知

在设置中配置：

- 哪些事件通知您
- 通知渠道（应用内、Telegram 等）
- 免打扰时段

## 获取帮助

### 应用内帮助

- **工具提示** — 悬停在图标和按钮上
- **空状态** — 部分为空时的有用提示
- **上下文提示** — 基于您正在做什么的提示

### 本文档

- 从导航栏链接的完整指南（如果启用）
- 搜索本文档
- 查看特定功能指南

## 常见工作流

### 开始新的一天

1. 检查 Overview 了解系统状态
2. 审查通知
3. 浏览实时动态了解夜间活动
4. 检查 Tasks 了解紧急项目
5. 回应任何人工审查项目

### 创建新公司

1. 点击导航栏中的 Companies
2. 点击 "New Company" 按钮
3. 填写简报表单
4. 提交并等待 CEO 计划
5. 审查并批准计划

### 检查 Agent 工作

1. 点击导航栏中的 Tasks
2. 按负责人或状态筛选
3. 审查 Review 列中的任务
4. 批准或请求更改
5. 检查 Activity 标签了解详情

### 监控系统健康

1. 检查 Overview 面板指标
2. 验证 Gateway 状态为绿色
3. 审查错误计数
4. 如果有问题，检查 Logs
5. 根据需要调整设置

## 效率技巧

1. **使用搜索** — Cmd/Ctrl + K 比导航更快
2. **观看实时动态** — 实时查看活动
3. **键盘快捷键** — 学习常用的
4. **固定重要内容** — 保持关键 Agent/公司已添加书签
5. **通知** — 配置以了解重要事项
6. **定期检查** — 每天审查仪表板

## 故障排除

**页面无法加载：**

- 检查 CapiBot 核心是否正在运行（端口 3000）
- 验证 Mission Control 端口（3002）
- 清除浏览器缓存

**网关断开连接：**

- 检查 WebSocket 端口 3001
- 验证防火墙设置
- 检查 CORS 配置

**性能缓慢：**

- 检查数据库连接
- 审查活跃会话数
- 清除旧日志和活动
