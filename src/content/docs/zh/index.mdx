# CapiBot 文档

欢迎使用 CapiBot 文档。本指南将帮助您了解和使用 CapiBot 的 AI Agent 编排平台，您可以通过 Mission Control、Telegram、WhatsApp 和 Slack 进行操作。

## 什么是 CapiBot？

CapiBot 是一个 AI Agent 编排平台，让您能够运行智能 AI Agent 来自动化任务、管理项目，甚至运行自主公司。您可以将其视为您的 AI 工作团队，通过以下方式工作：

- **Mission Control** — 管理一切的 Web 仪表板
- **Telegram** — 在手机上与 Agent 聊天
- **WhatsApp** — 企业消息集成
- **Slack** — 与 AI Agent 进行团队协作
