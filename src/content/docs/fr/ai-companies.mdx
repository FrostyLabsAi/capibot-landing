# Entreprises IA

Les Entreprises IA sont la fonctionnalité la plus puissante de CapiBot. Elles vous permettent d'exécuter des organisations autonomes composées entièrement d'agents IA, avec autant ou aussi peu de supervision humaine que vous le souhaitez.

## Qu'est-ce qu'une Entreprise IA ?

Une Entreprise IA est un groupe d'agents qui travaillent ensemble vers des objectifs communs, dirigés par un agent CEO :

- **Agent CEO** — Crée des plans, embauche des membres d'équipe, assigne le travail
- **Team Agents** — Spécialistes qui exécutent les tâches
- **Shared Goals** — Objectifs communs vers lesquels tout le monde travaille
- **Knowledge Base** — Référentiel central des informations de l'entreprise
- **Budget Tracking** — Suivi des coûts dans toute l'entreprise

Pensez-y comme une startup, mais où les employés sont des agents IA.

## Cycle de Vie de l'Entreprise

Les entreprises progressent à travers cinq phases :

### 1. Briefing

Vous décrivez ce que vous voulez que l'entreprise fasse :

- Nom et mission de l'entreprise
- Objectifs et buts initiaux
- Contraintes budgétaires
- Mode de gestion (Managed ou Autonomous)

### 2. Planning

L'agent CEO crée un plan d'affaires détaillé :

- Approche stratégique pour atteindre les objectifs
- Structure d'équipe et rôles nécessaires
- Calendrier et jalons
- Exigences en ressources

**Vous révisez et approuvez** avant de poursuivre.

### 3. Approved

L'entreprise est prête à opérer :

- Le plan d'affaires est approuvé
- Le CEO peut commencer à embaucher des membres d'équipe
- Le budget est alloué
- Les objectifs sont activés

### 4. Operating

L'entreprise fonctionne de manière autonome :

- Le CEO embauche des agents depuis le roster
- Les tâches sont créées et assignées
- Le travail circule dans le tableau Kanban
- Les livrables sont produits

### 5. Archived

Les opérations de l'entreprise se terminent :

- Toutes les tâches terminées ou transférées
- Base de connaissances préservée
- Résumé final des coûts
- Option de réactiver plus tard

## Modes de Gestion

### Managed Mode (Recommandé pour les Nouveaux Utilisateurs)

En mode Managed, vous restez maître :

- **Business plans** nécessitent votre approbation
- **Agent hiring** nécessite votre validation
- **Key tasks** vont dans la colonne Human Review
- **Budget changes** doivent être approuvées

**Idéal pour :** Apprendre le système, projets importants, quand vous voulez une supervision

### Autonomous Mode

En mode Autonomous, le CEO fonctionne indépendamment :

- Prend des décisions dans le cadre du plan approuvé
- Embauche des agents selon les besoins
- Exécute des tâches sans interruption
- Rend compte de l'avancement périodiquement

**Idéal pour :** Opérations de routine, utilisateurs expérimentés, automatisation sans intervention

## Quand Utiliser des Entreprises IA vs Agents Individuels

### Utiliser des Entreprises IA Quand :

| Scénario | Pourquoi Utiliser une Entreprise |
| --------------------------- | ------------------------------------------- |
| Campagnes marketing mensuelles | Processus cohérent, même équipe chaque mois |
| Support client continu | Agents support dédiés, connaissances partagées |
| Développement produit | Engineers, designers, QA travaillant ensemble |
| Production de contenu | Writers, éditeurs, chercheurs collaborant |
| Recherche de marché | Équipe de recherche avec méthodologie définie |

### Utiliser des Agents Individuels Quand :

| Scénario | Pourquoi Utiliser des Agents Individuels |
| -------------------------- | -------------------------------------- |
| Question de recherche ponctuelle | Simple, ne nécessite pas de coordination |
| Contenu rapide | Plus rapide que de configurer une entreprise |
| Exploration d'une idée | Tester avant de s'engager dans une entreprise |
| Tâches d'assistant personnel | Contrôle direct sur des tâches simples |

## Créer Votre Première Entreprise IA

### Étape 1 : Ouvrir le Panneau Entreprises

Dans Mission Control, cliquez sur **Entreprises** dans la barre latérale gauche.

### Étape 2 : Créer une Nouvelle Entreprise

Cliquez sur le bouton **"Nouvelle Entreprise"** et remplissez :

- **Nom** — ex. "Équipe Marketing Q1"
- **Mission** — Ce que l'entreprise accomplira
- **CEO** — Sélectionnez parmi les agents CEO disponibles (comme Atlas)
- **Management Mode** — Commencez par "Managed" pour plus de sécurité

### Étape 3 : Définir les Objectifs

Ajoutez des objectifs spécifiques et mesurables :

- "Lancer 4 articles de blog ce mois"
- "Générer 50 leads qualifiés"
- "Créer une vidéo de démonstration produit"

### Étape 4 : Soumettre le Briefing

Le CEO reçoit votre briefing et commence la planification.

### Étape 5 : Réviser le Plan

Quand le CEO soumet un plan d'affaires :

1. Révisez l'approche
2. Vérifiez la structure d'équipe proposée
3. Contrôlez le calendrier et le budget
4. Cliquez sur **Approuver** ou demandez des modifications

### Étape 6 : Surveiller les Opérations

Une fois approuvé :

- Observez les tâches apparaître sur le tableau Kanban
- Révisez les livrables dans l'onglet Knowledge
- Approuvez les demandes d'embauche au fur et à mesure
- Discutez avec l'entreprise dans l'onglet Chat

## Comprendre les Composants de l'Entreprise

### Organigramme

Arbre visuel montrant la structure hiérarchique :

```
CEO (Atlas)
├── Marketing Strategist (Echo)
│   └── Content Writer (Nova)
├── Designer (Pixel)
└── Research Analyst (Scout)
```

### Base de Connaissances

Informations spécifiques à l'entreprise :

- **Executive Summary** — Mission et stratégie (injecté dans le contexte de tous les agents)
- **Links** — URLs importantes et références
- **Guides** — Documents et procédures
- **Deliverables** — Travail terminé sauvegardé automatiquement ici

### Suivi des Objectifs

Chaque objectif montre :

- Statut (Active, Achieved, Archived)
- Indicateurs de progrès
- Tâches connexes
- Date d'achèvement

### Surveillance des Coûts

Suivez les dépenses dans toute l'entreprise :

- Répartition des coûts par agent
- Utilisation des tokens par modèle
- Tendances mensuelles
- Budget vs réel

## Travailler avec Votre Entreprise IA

### Via Mission Control

- **Onglet Overview** — Statut de l'entreprise, phase et statistiques
- **Onglet Plan** — Voir et éditer le plan d'affaires
- **Onglet Team** — Organigramme et workflow d'embauche
- **Onglet Tasks** — Tableau Kanban spécifique à l'entreprise
- **Onglet Knowledge** — Documents et livrables
- **Onglet Chat** — Conversation directe avec les agents de l'entreprise
- **Onglet Activity** — Tout ce que l'entreprise a fait
- **Onglet Approvals** — Décisions en attente nécessitant votre validation
- **Onglet Costs** — Analyse des dépenses

### Via les Applications de Messagerie

Envoyez des messages à votre entreprise :

```
Vous : @Q1 Marketing Team Comment va l'article de blog ?
CEO : Le Content Writer a terminé le brouillon.
      Il est en statut Review. Voulez-vous le voir ?
```

## Workflows Courants

### Lancer une Campagne Marketing

1. Créer l'entreprise "Campagne Mars" avec Atlas comme CEO
2. Définir les objectifs : "Générer 100 leads" et "5 articles de blog"
3. Approuver le plan du CEO
4. Le CEO embauche Marketing Strategist et Content Writer
5. Les tâches apparaissent automatiquement sur le tableau
6. Réviser les brouillons dans l'onglet Knowledge
7. Approuver le contenu final
8. Archiver l'entreprise après la fin de la campagne

### Exécuter le Support Client

1. Créer l'entreprise "Équipe Support" avec statut permanent
2. Le CEO embauche des agents Support
3. Connecter à vos canaux de support
4. Les agents gèrent les tickets comme des tâches
5. Construire la base de connaissances à partir des problèmes résolus
6. Réviser les rapports de performance mensuels

## Changer de Mode

Vous pouvez changer de mode de gestion à tout moment :

**Managed → Autonomous :**

- Quand vous êtes confiant dans les décisions du CEO
- Pour les opérations de routine en qui vous avez confiance
- Après que l'équipe a fait ses preuves

**Autonomous → Managed :**

- Quand les décisions importantes nécessitent un examen
- Si vous voulez changer de direction
- Pour réviser avant des jalons critiques

## Conseils pour Réussir

1. **Commencez avec le Mode Managed** — Apprenez comment votre CEO prend des décisions avant de passer en autonomous
2. **Définissez des Objectifs Clairs** — Des objectifs spécifiques et mesurables aident les agents à se concentrer
3. **Révisez les Plans Attentivement** — Le plan du CEO pose les bases de tout
4. **Construisez la Base de Connaissances** — Sauvegardez les documents importants pour référence future
5. **Surveillez au Début** — Observez attentivement les premières tâches, puis réduisez la supervision
6. **Archivez Quand C'est Terminé** — Gardez votre tableau de bord propre, vous pouvez réactiver plus tard
