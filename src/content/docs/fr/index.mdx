# Documentation CapiBot

Bienvenue dans la documentation CapiBot. Ce guide vous aidera à comprendre et à utiliser la plateforme d'orchestration d'agents IA CapiBot via Mission Control, Telegram, WhatsApp et Slack.

## Qu'est-ce que CapiBot ?

CapiBot est une plateforme d'orchestration d'agents IA qui vous permet d'exécuter des agents IA intelligents pour automatiser des tâches, gérer des projets et même diriger des entreprises autonomes. Pensez-y comme votre main-d'œuvre IA qui travaille via :

- **Mission Control** — Un tableau de bord web où vous gérez tout
- **Telegram** — Discutez avec des agents sur votre téléphone
- **WhatsApp** — Intégration de messagerie professionnelle
- **Slack** — Collaboration d'équipe avec des agents IA
