### Badges de Phase

Indicateurs visuels de l'état de l'entreprise :

- 🟡 **Briefing** — Configuration en cours, en attente de plan
- 🔵 **Planning** — CEO crée le plan d'affaires
- 🟢 **Approved** — Plan approuvé, prêt à opérer
- 🟣 **Operating** — Actuellement en cours d'exécution
- ⚪ **Archived** — Terminé ou en pause

## Créer une Entreprise

### Étape par Étape

1. **Cliquez "+ Nouvelle Entreprise"**

2. **Remplissez les Détails de l'Entreprise :**

   ```
   Nom: Q1 Marketing Campaign
   Mission: Lancer une campagne de sensibilisation produit
            ciblant les clients entreprise

   CEO: [Atlas ▼]
   Mode: [Managed ▼]
   Budget: $100/month
   ```

3. **Définissez les Objectifs :**

   ```
   Objectif 1: Publier 4 articles de blog
   Objectif 2: Générer 50 leads qualifiés
   Objectif 3: Créer une vidéo de démo produit
   ```

4. **Soumettez le Briefing**

5. **CEO Crée le Plan** (généralement dans les minutes)

6. **Révisez le Plan** — Voir section suivante

## Changer les Modes de Gestion

### Managed → Autonomous

**Quand prêt :**

1. Allez dans l'onglet Overview de l'entreprise
2. Cliquez sur "Mode: Managed" dropdown
3. Sélectionnez "Autonomous"
4. Confirmez : "Êtes-vous sûr ? Le CEO prendra des décisions sans approbation."

**Le CEO peut alors :**

- Embaucher des agents sans demander
- Créer des tâches automatiquement
- Exécuter des plans indépendamment
- Rendre compte de l'avancement périodiquement

### Autonomous → Managed

**Quand vous voulez une supervision :**

1. Allez dans l'onglet Overview de l'entreprise
2. Cliquez sur "Mode: Autonomous" dropdown
3. Sélectionnez "Managed"
4. Toutes les futures décisions nécessiteront une approbation

## Archiver des Entreprises

Quand une entreprise termine sa mission :

1. **Cliquez le bouton "Archive"**
2. **Choisissez l'option :**
   - **Archive Only** — Préserve tout, arrête les opérations
   - **Archive & Transfer** — Déplace les tâches inachevées ailleurs
   - **Delete** — Supprime complètement (utilisez avec précaution !)

3. **L'entreprise passe à l'état Archived**
   - Agents libérés (peuvent être réutilisés)
   - Tâches gelées
   - Base de connaissances préservée
   - Peut être réactivée plus tard

## Bonnes Pratiques

### Configuration des Entreprises

1. **Mission Claire** — Objectifs spécifiques et réalisables
2. **Bon CEO** — Choisir basé sur l'expertise domaine
3. **Budget Réaliste** — Commencer petit, augmenter
4. **Calendrier Défini** — 2-8 semaines typiquement
5. **Objectifs Mesurables** — Cibles spécifiques

### Opération des Entreprises

1. **Révisez les Plans Attentivement** — Pose les bases de tout
2. **Surveillez au Début** — Observez les premières tâches attentivement
3. **Construisez la Base de Connaissances** — Sauvegardez les documents importants
4. **Vérifiez les Coûts Hebdomadairement** — Évitez les surprises
5. **Communiquez Régulièrement** — Utilisez l'onglet Chat pour les mises à jour

### Approbation du Travail

1. **Révisions en Temps Opportun** — Ne laissez pas les tâches s'attarder dans Human Review
2. **Feedback Clair** — Demandes de modifications spécifiques
3. **Documentez les Décisions** — Ajoutez des notes aux approbations
4. **Faites Confiance mais Vérifiez** — Contrôlez ponctuellement le travail autonomous

## Workflows Courants

### Lancer une Campagne

1. Créer l'entreprise "Campagne Mars"
2. Assigner Atlas comme CEO
3. Définir les objectifs : "Générer 100 leads"
4. Atlas soumet le plan avec structure d'équipe
5. Approuver le plan
6. Atlas embauche Echo (Strategy), Nova (Content), Pixel (Design)
7. Les tâches apparaissent sur le tableau
8. Réviser les livrables dans l'onglet Knowledge
9. Approuver le contenu final
10. Archiver l'entreprise quand terminé

### Exécuter du Support Continu

1. Créer l'entreprise "Équipe Support"
2. Définir le statut permanent (pas de date de fin)
3. Atlas embauche des agents Support
4. Connecter aux canaux de support
5. Les agents gèrent les tickets comme des tâches
6. Construire la base de connaissances à partir des solutions
7. Réviser les analyses mensuelles

## Dépannage

**Entreprise Bloquée en Planning :**

- Le CEO peut avoir besoin de plus d'informations
- Vérifiez l'onglet Chat pour les questions
- Fournissez des clarifications
- Ou éditez le briefing et soumettez à nouveau

**Tâches qui Ne Bougent Pas :**

- Vérifiez si les agents sont Online
- Vérifiez que les tâches ne sont pas Blocked
- Regardez l'onglet Activity pour les erreurs
- Peut avoir besoin de générer plus d'agents

**Coûts Trop Élevés :**

- Révisez l'onglet Costs pour la répartition
- Envisagez des modèles LLM moins chers
- Réduisez le nombre d'agents
- Définissez des limites de budget plus strictes

**Approbations qui S'Accumulent :**

- Révisez régulièrement l'onglet Approvals
- Prévoyez du temps pour les révisions
- Envisagez de passer en Autonomous
- Ou embauchez moins d'agents
