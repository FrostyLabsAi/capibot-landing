1. Provedor primário (Claude) tenta requisição

```
Traces            Manter por [30 ▼] dias

☑️ Habilitar limpeza automática
☐ Arquivar antes de excluir
[Salvar Mudanças]
```

**Por Que Limpeza Automática?**

- Reduz tamanho do banco de dados
- Melhora performance
- Mantém privacidade
- Controla custos

**Dados Importantes Nunca São Excluídos:**

- Tarefas ativas
- Memórias de agentes
- Base de conhecimento
- Informação de empresa
- Configurações

## Boas Práticas

### Configuração

1. **Documente Mudanças** — Note por que você mudou configurações
2. **Teste Primeiro** — Tente mudanças em ambiente de dev
3. **Backup Primeiro** — Antes de mudanças maiores
4. **Comece Conservador** — Aperte gradualmente
5. **Revise Regularmente** — Auditoria de configurações mensal

### Segurança

1. **Chaves de API Fortes** — Use chaves auto-geradas
2. **Rotacione Regularmente** — A cada 90 dias
3. **Menor Privilégio** — Acesso mínimo necessário
4. **Monitore Logs** — Observe atividade incomum
5. **Mantenha Atualizado** — Última versão para patches de segurança

### Performance

1. **Pool do Tamanho Certo** — Combine conexões de DB ao uso
2. **Cache com Sabedoria** — Não cache conteúdo dinâmico
3. **Limpe Regularmente** — Limpeza automática de dados antigos
4. **Monitore Recursos** — Observe memória e disco
5. **Otimize Queries** — Verifique logs de query lenta

## Solução de Problemas

**Configurações Não Salvando:**

- Verifique permissões
- Verifique se valores são válidos
- Revise mensagens de erro
- Tente atualizar página

**Teste de Conexão Falha:**

- Verifique chaves de API
- Verifique conectividade de rede
- Revise regras de firewall
- Verifique status do serviço

**Problemas de Performance:**

- Diminua tamanho do pool de conexões
- Reduza duração do cache
- Habilite limpeza automática
- Verifique tamanho do banco de dados

**Alertas de Segurança:**

- Revise logs de auditoria
- Verifique padrões de acesso
- Verifique listas de permissões
- Rotacione chaves comprometidas
