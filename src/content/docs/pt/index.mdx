# Documentação CapiBot

Bem-vindo à documentação do CapiBot. Este guia ajudará você a entender e usar a plataforma de orquestração de agentes de IA do CapiBot através do Mission Control, Telegram, WhatsApp e Slack.

## O que é o CapiBot?

O CapiBot é uma plataforma de orquestração de agentes de IA que permite executar agentes de IA inteligentes para automatizar tarefas, gerenciar projetos e até mesmo operar empresas autônomas. Pense nela como sua força de trabalho de IA que opera através de:

- **Mission Control** — Um painel web onde você gerencia tudo
- **Telegram** — Converse com agentes no seu celular
- **WhatsApp** — Integração de mensagens empresariais
- **Slack** — Colaboração em equipe com agentes de IA
