## Panel de Análisis

Métricas de rendimiento y tendencias.

**Acceso:** Barra de navegación → **Analytics**

### Métricas del Panel

**Rendimiento de Agentes:**

- Tareas completadas por agente
- Tiempo promedio de completitud
- Tasas de éxito/error
- Costo por agente

**Métricas de Tareas:**

- Tareas por estado
- Velocidad de completitud
- Tiempo promedio en cada columna
- Identificación de cuellos de botella

**Métricas de Sistema:**

- Volumen de llamadas a API
- Rendimiento de base de datos
- Conexiones WebSocket
- Uso de memoria

### Rangos de Tiempo

Ver datos para:

- Últimas 24 horas
- Últimos 7 días
- Últimos 30 días
- Rango personalizado

### Exportar

Descarga reportes como:

- CSV (hojas de cálculo)
- JSON (datos crudos)
- PDF (reportes formateados)

## Audit Trail

Logging de seguridad y cumplimiento.

**Acceso:** Barra de navegación → **Audit Trail**

### Qué se Registra

Cada acción con:

- **Who** — Usuario o agente
- **What** — Acción realizada
- **When** — Marca de tiempo
- **Where** — Dirección IP
- **How** — User agent (navegador/app)

**Categorías de Acción:**

- Autenticación (login, logout)
- Operaciones de tareas (crear, actualizar, eliminar)
- Operaciones de agentes (generar, archivar)
- Cambios de configuración
- Acciones de integración
- Eventos de webhook

## Mejores Prácticas

### Monitoreo

1. **Revisa Diariamente** — Revisa actividad y métricas
2. **Observa Patrones** — Detecta tendencias temprano
3. **Configura Alertas** — Recibe notificación de issues
4. **Revisa Semanalmente** — Tendencias de análisis

### Debugging

1. **Comienza con Logs** — Encuentra mensajes de error
2. **Revisa Traces** — Ve qué hicieron los agentes
3. **Revisa Activity** — Línea de tiempo de eventos
4. **Audit Trail** — Quién hizo qué cuándo

### Rendimiento

1. **Monitorea Latencia** — Tiempos de respuesta de agentes
2. **Observa Costos** — Uso de tokens por agente
3. **Rastrea Tasas de Éxito** — Métricas de calidad
4. **Optimiza Cuellos de Botella** — Operaciones lentas

## Solución de Problemas

**No Muestra Actividad:**

- Verifica que el sistema esté ejecutándose
- Verifica que los filtros no sean demasiado restrictivos
- Refresca la página

**Logs No se Actualizan:**

- Habilita modo "Live"
- Revisa filtros de nivel de log
- Puede necesitar desplazarse

**Notificaciones Faltantes:**

- Revisa configuración de notificaciones
- Verifica conexiones de canales (Telegram, etc.)
- Revisa configuración de "No Molestar"

**Costos Altos:**

- Revisa Análisis para desglose
- Revisa traces para operaciones costosas
- Considera selección de modelo
